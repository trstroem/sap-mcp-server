# 🚀 How to Run SAP MCP

## 🖥️ Run the Server

1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Start the server:
   ```sh
   python sap-mcp-server.py
   ```

## 💻 Run the Client

1. Change to the `test` directory:
   ```sh
   cd test
   ```
2. Start the client:
   ```sh
   python sap-mcp-client.py
   ```
